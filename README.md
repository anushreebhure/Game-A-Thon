### Problem Statement 1 :Given Two Integers Write a Function to print Factorial of a number
#### TestCase 1: Input : 5 , Output : 120
```python
def factorial(n):
    if n==1 or n==0:
        return 1
    else:
        return n*factorial(n-1)
n=int(input())
print(factorial(n))
```
